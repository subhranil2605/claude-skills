# 🚀 Engineering Standards — Claude Skill

AI can generate code.

But it doesn’t guarantee:
- security
- scalability
- maintainability
- production readiness

This repository contains a **Claude Skill** that fixes that.

---

## 🧠 What is this?

**Engineering Standards** is a reusable Claude Skill that acts like an:

> 🧠 AI Tech Lead for your code

It enforces best practices across:
- Security
- Performance
- Code Quality
- Reliability
- DevOps
- Compliance

---

## ⚠️ Problem

With vibe coding / AI-generated code:

- Output is inconsistent  
- Security is often ignored  
- Architecture is weak  
- Bugs go unnoticed  

👉 Result: tech debt, failures, and unsafe systems

---

## ✅ Solution

This skill:

- Reviews AI-generated code
- Guides architecture decisions
- Enforces engineering standards
- Prevents common mistakes

---

## 💡 Use Cases

- Fixing AI-generated code (Claude, ChatGPT, etc.)
- SaaS development
- Backend/API design
- Code reviews
- Training junior developers
- AI workflows (n8n, agents)

---

## ⚙️ How to Use

### 1. Download the skill
Clone this repo:

```bash
git clone https://github.com/your-username/engineering-standards.git
````

---

### 2. Upload to Claude

* Open Claude
* Go to **Skills**
* Upload the `engineering-standards/` folder

---

### 3. Use it

#### In Claude Chat:

Ask:

```
Is this API design correct?
```

```
Review this code for security issues
```

#### In Claude Code:

The skill automatically:

* reviews your code
* guides decisions
* enforces standards
